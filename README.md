 🌤️ Nimbus Weather App
 
A simple weather app built with HTML, Tailwind CSS and JavaScript.
 
## Live Demo
[View Live](https://yourusername.github.io/weather-app)
 
## Built With
- HTML, Tailwind CSS, JavaScript
- OpenWeather API
 
## Setup
1. Clone the repo
2. Create a `config.js` file and add your API key:
```javascript
const config = {
  apiKey: "your_api_key_here"
}
```
3. Open `index.html` in your browser
